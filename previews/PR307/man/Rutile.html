Rutile ​

To obtain the list of implemented diffusion parameters for rutile, use:

GeoParams.MaterialParameters.ChemicalDiffusion.Rutile.chemical_diffusion_list Function
julia
chemical_diffusion_list(search::String="")

List all available chemical diffusion data for rutile. Includes an argument to search for a specific term, i.e. an element ("La") or an author.

source

The following diffusion parameters for rutile are implemented:
