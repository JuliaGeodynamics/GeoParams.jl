Radioactive heat

Methods

Radioactive heat sources are defined as

GeoParams.MaterialParameters.RadioactiveHeat.ConstantRadioactiveHeat — Type
ConstantRadioactiveHeat(H_r=1e-6Watt/m^3)

Set a constant radioactive heat:

\[ H_r = cst\]

where $H_r$ is the radioactive heat source [$Watt/m^3$].

source

Computational routines

To compute, use this:

GeoParams.MaterialParameters.RadioactiveHeat.compute_radioactive_heat — Function
H_r = compute_radioactive_heat(s:<AbstractRadioactiveHeat)

Returns the radioactive heat H_r
